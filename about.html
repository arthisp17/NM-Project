About This Project

A modern, elegant E-Book Management System with PDF Support

Features

Add, view, and delete books
Upload PDF files for each book
Read PDF books with built-in PDF viewer
Search books by title or author
Favorite books for quick access
Filter by genre
Persistent local storage (IndexedDB for PDFs)
Beautiful glassmorphism design

Technologies

HTML5, CSS3, JavaScript (ES6), LocalStorage, IndexedDB, PDF.js, Font Awesome
